A TAG body resumidamente serve para dar ao site as caracteristicas vizuais.
A TAG style serve para abordar todas as caracteristicsa complementares para o dsing do visual criado pela TAG body.
A TAG strong serve para tornar as palavras e frases em negrito .
A TAG B tem como funçao a mesma que a da tag STRONG ,que é tronar as palavras em negrito
A TAG IMG complementada ela SRC sao usadas para colocar vizualmete na pagina uma imagem, e com o complemento SRC voce identifica na pasta salva a imagem desejada apos salvarla na mesma pasta que a pagina do site index
